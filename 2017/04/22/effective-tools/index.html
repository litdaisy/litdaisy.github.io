学习工具

高效学习工作的神兵利器

Chrome

chrome 浏览器具有强大的扩展功能，可以添加各类扩展，使平时学习工作更加高效；而且 chrome 界面美观，配合翻墙使用 Google 搜索引擎，你能更加准确搜到你所需要的。

英语类

有道词典

网易出品的产品，素以界面美观优雅著称，比如网易云音乐。新版有道词典 PC 端一改之前淡蓝风格，红色更显活力，Windows 系统有这么漂亮的软件实属少见。更重要的是，有道词典的划词和取词功能也非常好用，使你可以快速地阅读外文文献或者英文开发文档。

另外，配合手机端有道词典，以及 chrome 的扩展有道划词插件，各种场合碰到的生涩词汇一网打尽。

Google 翻译

Google 在自然语言识别、翻译和语音识别等方面深有造诣，但很多时候都需要翻墙才能使用 Google 的产品。不过，最近推出的 Google 翻译完全不需要翻墙就可以毫无障碍地使用。其实景翻译功能可谓令人汗颜，用 Google 翻译手机端，再也不需要担心出行时不识英文路牌了。

笔记写作类

印象笔记

印象笔记最大的优点在于，它可以把你在各个平台看到的有趣的文章或网页一件保存到你的笔记，比如 chrome 上浏览到的网页，微信公众号推送的文章（相比而言，微信自带的收藏功能就显得很鸡肋了，试问你收藏在微信的文章，毫无分类可言，找都不找到，还怎么去回顾）。配合印象笔记 PC 端，印象笔记手机端和 chrome 印象笔记扩展插件，实现笔记多平台无缝保存和同步。

Typora

如果说世界上最好的编程语言是 PHP，没有之一。那么我认为，世界上最好的 Markdown 编辑器便是 Typora，同样没有之一。Typora 相比于传统 Markdown 编辑器最大的不同在于：即写即视。就是说，文本编辑界面和最终预览界面整合为一，写到哪，就预览到哪。另外，还有其极简主义的风格看起来非常舒服，内置多种主题任你选择。关键这么好用的 Markdown 编辑器，目前无论在 Windows 下还是 XOS 下，都是免费的。

Typora，一个让你爱上写作的 Markdown 编辑器。

Notepad++

强大的文档编辑器，以后就不要用 Windows 的记事本了。

博客类

七牛云

博客写作最烦人的是如何处理图片，你可能经常会将一篇博客发布到不同平台（如 CSDN，简书等），但是你能忍受每发一遍都要重新将博客中图片视频等资源重新上传一次吗？这种低效工作当然是要避免的。七牛云作为一个云存储，为你免费提供一个 10G 大的云空间，你可以将其作为你博客的云上图传，从而做到图片只传一次，多博客平台共享资源。

另外，七牛云还支持将资源外链的前缀更改为自己的域名（只支持国内备案的域名），是不是很酷炫。七牛云还提供了云上图片处理功能，你只需要做的是，在外链上更改参数，就能直接获取处理后的图片。

极简图床

极简图床是一个 chrome 的扩展程序，将图片往里一拖，快速上传并返回图片外链及 Markdown 图片格式的链接，以供复制，让传图更加的快捷高效。关键是，极简图床还能关联七牛云空间，快捷容量两不误。

开机必自启类

Listary

不只是一个强大的文件搜索软件。它的 ctrl+ctrl 和 ctrl+G 快捷键，不知道为我省下多少时间。

FSCapture

不只是一个强大的截图软件，它还是一个录屏、图像处理、视频编辑器，屏幕标尺、取色器、放大器等，功能远不止于此。
